# 17. Recursion
#### E.g.) Factorial
- Algorithm)
  ```python
  def factorial(n):
    if n == 0 or n == 1:
      return 1
    
    return factorial(n-1) * n
  ```
- Pf.)
  - Base Case)
    - `factorial(0)`$`= 1 = 0!`$
    - `factorial(1)`$`= 1 = 1!`$
  - Suppose the algorithm holds for $`n=k`$.
    - Then `factorial(k)`$`=k!`$
  - Now consider `factorial(k+1)`.
    - The algorithm will return `factorial(k) * (k+1)`$`=k! * (k+1) = (k+1)!`$.
  - By induction, the algorithms works for $`\forall n \in \mathbb{Z}^+`$.
